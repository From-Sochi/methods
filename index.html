МЕТОДЫ

ЧИСЛО СТРОКА МАССИВ ОБЪЕКТ

Math.pow()

Возводит число в заданную степень.

Math.pow(число, степень)

.toUpperCase()

Преобразование строки в верхний регистр. При этом возвращается новая строка, а исходная строка не меняется.

строка.toUpperCase();

.join()

Объединяет элементы массива в строку с указанным разделителем (он будет вставлен между элементами массива). Разделитель задается параметром метода и не является обязательным. Если он не задан - по умолчанию в качестве разделителя возьмется запятая. Вернет строку

массив.join([разделитель]);

Object.keys()

Ввозвращает массив из свойств объекта в том же порядке, как и при перечислении через цикл.

let values = Object.keys(объект, из которого выводим значения);

Math.sqrt()

Возвращает квадратный корень числа. Вернет число

Math.sqrt(положительное число);

.toLowerCase()

Преобразует символы строки в нижний регистр. При этом мы получаем новую строку, а исходная строка остается неизменной.

строка.toLowerCase();

.shift()

Удаляет первый элемент из массива. При этом исходный массив изменяется, а результатом метода возвращается удаленный элемент.

массив.shift();

Object.values()

Возвращает массив из значений объекта в том же порядке, как и перечислении через цикл.

let values = Object.values(объект, из которого выводим значения);

Math.round()

Округляет до ближайшего целого числа по правилам математического округления.

Math.round(число);

.substr()

Возвращает подстроку из строки (исходная строка при этом не изменяется).

строка.substr(откуда отрезать, [сколько символов отрезать]);

.pop()

Удаляет последний элемент из массива. При этом исходный массив изменяется, а результатом метода возвращается удаленный элемент.

массив.pop();

Object.assign()

Копирует свойства и значения исходного объекта, возвращая при этом новый объект. В первом параметре метода мы задаем целевой объект, в который копируем, а во втором - указываем объекты, которые нужно скопировать.

Object.assign(куда копировать, объекты);

Math.ceil()

Округление дробного числа до целого всегда в большую сторону.

Math.ceil(число);

.substring()

Возвращает подстроку из строки (исходная строка при этом не изменяется).

строка.substring(откуда начать отрезать, [докуда отрезать]);

.push()

Добавляет неограниченное количество элементов в конец массива. При этом исходный массив изменяется, а результатом возвращается новая длина массива.

массив.push(элемент, элемент, элемент...);

Math.floor()

Округление числа до целых всегда в меньшую сторону.

Math.floor(число);

.slice()

Возвращает вырезанную подстроку из строки (исходная строка при этом не изменяется).

строка.slice(откуда отрезать, [докуда отрезать]);

.unshift()

Добавляет неограниченное количество новых элементов в начало массива. При этом исходный массив изменяется, а результатом возвращается новая длина массива.

массив.unshift(элемент, элемент, элемент...);

.toFixed()

Округление числа до указанного знака в дробной части.

число.toFixed([количество знаков в дробной части]);

.includes('')

Поиск заданной строки в текущей с учетом регистра. Вернет булевое значение

строка.includes('что ищем', [откуда начинать поиск]);

.slice()

Вырезает и возвращает указанную часть массива. Сам массив при этом не изменяется.

массив.slice(откуда отрезать, [докуда отрезать]);

.toPrecision()

Округляет число до заданного знака.

число.toPrecision(длина);

.startsWith('')

Проверяет начинается ли строка с указанной в первом параметре подстроки. Вернет булевое значение

строка.startsWith('что ищем', [начало проверки]);

.splice( , )

Удаляет или добавляет элементы в массив. Можно только удалять элементы, только добавлять или делать и то и другое одновременно.

массив.splice(откуда удаляем, сколько удаляем, [вставить], [вставить]...);

Math.max()

Возвращает максимальное число из группы чисел.

Math.max(число, число, число...);

.endsWith('')

Проверяет, заканчивается ли строка на указанную в первом параметре подстроку. Вернет булевое значение

строка.endsWith('что ищем', [длина строки]);

.reverse()

Изменяет порядок элементов в массиве на обратный. Метод изменяет исходный массив (он станет перевернутым) и возвращает также перевернутый массив (можно пользоваться и так, и так).

массив.reverse();

Math.min()

Возвращает минимальное число из группы чисел.

Math.min(число, число, число...);

.indexOf('')

Осуществляет поиск подстроки в строке. Метод вернет позицию первого совпадения, а если оно не найдено, то вернет -1.Вернет число

строка.indexOf('что ищем', [откуда начинать поиск]);

Math.random()

Возвращает случайное дробное число от 0 до 1.

Math.random();

.lastIndexOf('')

Поиск последнего вхождения подстроки в строке. Метод вернет позицию первого совпадения с конца, а если оно не найдено, то вернет -1.Вернет число

строка.lastIndexOf('что ищем', [откуда начинать поиск]);

Math.abs()

Возвращает модуль числа, то есть из отрицательного числа делает положительное.

Math.abs(число);

.replace('')

Gоиск и замену частей строки. Первым параметром принимается подстрока, которую заменяем, а вторым - подстрока, на которую заменяем.

строка.replace('что заменяем', 'на что заменяем');

.split()

Разбивает строки в массив по указанному в первом параметре разделителю. Если он не задан - вернется вся строка. Если он задан как пустые кавычки, то каждый символ строки попадет в отдельный элемент массива. Вернет массив строк

строка.split([разделитель], [максимальное количество элементов]);

.length

Позволяет узнать длину строки. Под длиной понимается количество символов в ней. Вернет число

строка.length;
